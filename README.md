# Tetrads
Interactive tetrahedron of JI tetrads with the ability to scale via common complexity models.  

Sounds, shapes, colors ... Go nuts!

  • Shift+Hover to play chords
  • Shift+Hover(Click Hold) to hold chord; next chord on release
  • S A T B keys to toggle the pivoting common tone between adjacent chords

Approaching this as a sort of 3D take off of [Sintel's very fun triangle instrument](https://sintel.website/posts/triangle.html) 

For those interested: [information on complexity models](https://en.xen.wiki/w/User:Sintel/Validation_of_common_consonance_measures)
