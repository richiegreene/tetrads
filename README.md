# Tetrads
Interactive tetrahedron of JI tetrads with the ability to scale via common complexity models.  

Sounds, shapes, colors ... Have at it!

Shift+Hover to play chords
Shift+Hover(Click Hold) to hold chord; next chord on release
S A T B keys to toggle the pivoting common tone between adjacent chords

Approaching this all as a sort of 4HE take off of Sintel's very fun 3HE instrument:
https://sintel.website/posts/triangle.html

Information on complexity models:
https://en.xen.wiki/w/User:Sintel/Validation_of_common_consonance_measures
